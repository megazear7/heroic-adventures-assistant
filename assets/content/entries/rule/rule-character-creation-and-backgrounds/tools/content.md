# Rule: Character Creation and Backgrounds

## Scope

Use this rule for complete character build legality at level 1: race, flaw, background, stat package, class, features, training, and starting gear.

## Consolidated Creation Flow

1. Choose race and apply race base stats, movement, health bonus, and creation effects.
2. Roll flaw and apply the penalty.
3. Roll background and apply stat bonus and health/heroic bonus profile.
4. Choose one stat increase package and apply to distinct stats.
5. Choose class and apply class creation health and class-specific creation effects.
6. Choose one heroic class feature (or feat where legal).
7. Choose starting training (expertise and weapon category).
8. Spend starting coins for equipment and finalize combat profile.
9. Calculate final health from all listed contributors.

## Core Constraints

- Stat package increases must target different stats.
- Feature choices must respect action-slot limits and class/race legality.
- Training choices affect both competence and downstream progression.

## Background and Flaw Integration

- Flaws always impose a real mechanical tradeoff.
- Background bonuses can influence heroic action economy and health scaling over advancement.

## Adjudication Notes

- Validate each step before moving to next to prevent cascading build errors.
- If random roll options produce weak synergy, allow legal selection between paired options as written.
- Keep a checklist audit to ensure no creation bonus is missed.
