# Rule: Weapons, Mastery, and Enchantments

## Scope

Use this rule for weapon profiles, weapon categories, mastery features, and enchantment interactions.

## Consolidated Rule

Weapon choice defines attack damage dice, range, handedness, initiative profile, and category-based training benefits. Mastery features add category-specific tactical options and action-type hooks.

## Core Weapon Adjudication

1. Identify weapon profile (size/type/range/damage/initiative).
2. Apply relevant training and category bonuses.
3. Resolve attack and damage using standard attack rules.
4. Apply mastery features if trigger and action type are legal.

## Mastery System

- Blade, axe, blunt, polearm, and bow mastery each grant unique feature packages.
- Many mastery features consume bonus/reaction windows and must respect action economy limits.
- Mastery utility often overlaps with positioning and opportunity mechanics.

## Dual-Wielding Integration

- Primary weapon controls baseline initiative bonus.
- Secondary weapon use is constrained by feature text and specific action permissions.

## Enchantment Integration

- Weapon enchantments should be evaluated for damage bypass, action compression, and control impact.
- Maintain reward pacing to avoid overpowering progression bands.

## Adjudication Notes

- Category identity matters for training and mastery; do not cross-apply unless text explicitly allows.
- Track once-per-trigger usage carefully for mastery reactions/bonus actions.
