# Rules

This folder contains thematic rule summaries extracted from the **Heroic Adventures 2nd Edition** rulebook. Each entry focuses on a specific rules topic for quick reference during play or prep.

## Usage

- Use `rules_list` to see all available rule entries.
- Use `rules_get` with an `entry-name` to retrieve a specific rule summary.

## Available Rules

| Entry Name | Description |
|---|---|
| rule-action-economy-and-rounds | Turn structure, round sequencing, and action type adjudication |
| rule-attacks-damage-and-defenses | Attack resolution, modifiers, damage, temporary health, and defenses |
| rule-character-creation-and-backgrounds | Creation pipeline, flaws, backgrounds, and starting configuration |
| rule-classes-and-class-identity | Class feature interpretation, class fantasy, and role differentiation |
| rule-consumables-loot-and-economy | Consumables, loot scales, guild services, and reward pacing |
| rule-dying-rest-and-recovery | Dying state rules, long rest handling, and recovery |
