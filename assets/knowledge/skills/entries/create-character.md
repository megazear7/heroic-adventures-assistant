# Create Character

## Purpose

Produce a legal, table-ready character from concept through starting loadout and action-ready play notes.

## When to Use

- A player is making a new character.
- A replacement character is needed quickly mid-campaign.
- A concept must be translated into legal race/class/background choices.

## Required Inputs

- Character concept and fantasy.
- Starting level and campaign constraints.
- Party composition and desired role.
- Preferences: race, class, playstyle, complexity tolerance.

## Workflow

1. Clarify concept into role priorities (damage, control, support, durability, utility).
2. Select legal race, flaw, background, and class.
3. Apply starting stats and all granted increases in correct order.
4. Choose level-appropriate class features, training/expertise, and starting proficiencies.
5. Assign starting equipment and verify compatibility with class/race/training requirements.
6. Build ability/action list with explicit action type labels (major, minor, bonus, reaction, heroic, opportunity when applicable).
7. Add concise tactics for turn-1, sustained turns, and low-resource fallback.

## Quality Checks

- All choices match current level gates.
- Prerequisites for features/gear are satisfied.
- No duplicate or conflicting selections.
- Core combat loop is playable with listed actions.

## Output Template

Return a compact character sheet with:
- Identity + concept summary.
- Race, flaw, background, class, level.
- Core stats and derived combat profile.
- Feature list by source and level.
- Equipment and consumables.
- Action list with action-type tags.
- 3-turn tactical quickstart.

## Missing Information Policy

If details are missing, choose conservative defaults aligned to party needs, state assumptions, and include 2 legal alternatives for easy swap.

## Rule Routing

- Prioritize character creation, races, classes, leveling, equipment, and action economy rule skills.
- If a dependency is uncertain, stop short of finalizing and mark the unresolved prerequisite.
