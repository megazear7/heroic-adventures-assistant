# Skills

This folder contains reusable skill and prompt entries for **Heroic Adventures 2nd Edition** campaign play, prep, and content generation. These include both workflow skills and quick-use prompt templates.

## Usage

- Use `skills_list` to see all available skill entries.
- Use `skills_get` with an `entry-name` to retrieve a specific skill or prompt.

## Available Skills

| Entry Name | Description |
|---|---|
| adventure-hook | Generate adventure hooks for campaign play |
| build-npc | Build usable NPCs with motivations, roleplay cues, and mechanical hooks |
| create-character | Build complete characters from concept to ready-to-play sheet |
| explain-rule | Explain a specific rule clearly and concisely |
| generate-adventure | Generate one-shot or campaign-arc adventures |
| level-up-character | Validate and optimize character advancement choices |
| level-up-check | Quick level-up validation check |
| loot-and-rewards | Produce rewards aligned to encounter difficulty and progression |
| make-encounter | Design balanced, tactical encounters with pacing controls |
| make-monster | Create monsters using role language and action economy |
| mass-battle | Run and adjudicate unit-scale battles |
| monster-from-theme | Create a monster from a thematic concept |
| npc-generator | Quick NPC generation for table use |
| quick-roll | Create on-the-fly rulings and roll procedures |
| run-session | Live-session support with turn adjudication and pacing |
| session-prep | Build practical session prep packets |
| treasure-drop | Generate treasure drops for encounters |
