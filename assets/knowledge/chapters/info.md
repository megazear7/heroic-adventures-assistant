# Chapters

This folder contains the full text of each chapter from the **Heroic Adventures 2nd Edition** rulebook.

## Usage

- Use `chapters_list` to see all available chapters.
- Use `chapters_get` with an `entry-name` to retrieve the full text of a specific chapter.

## Available Chapters

| Entry Name | Description |
|---|---|
| chapter-01-introduction | Introduction to the Heroic Adventures system |
| chapter-02-characters | Character creation overview and core attributes |
| chapter-03-races | Playable races and ancestry features |
| chapter-04-classes | Character classes and class features |
| chapter-05-spells | Spell lists, casting rules, and spell descriptions |
| chapter-06-equipment | Weapons, armor, gear, and economy |
| chapter-07-proteges | Protégé (companion/follower) rules |
| chapter-08-mass-combat | Mass combat and unit-scale battle rules |
| chapter-09-action-checks | Action checks, skill checks, and resolution mechanics |
| chapter-10-gameplay | Core gameplay loop, exploration, and encounter flow |
| chapter-11-game-masters-guide | GM advice, worldbuilding, and campaign tools |
| chapter-12-monster-manual | Monster entries, stat blocks, and encounter creatures |
