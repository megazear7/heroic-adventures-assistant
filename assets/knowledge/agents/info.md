# Agents

This folder contains role-focused agent definitions for **Heroic Adventures 2nd Edition**. Each agent represents a specialized persona for rules lookup, GMing, and content generation workflows.

## Usage

- Use `agents_list` to see all available agent entries.
- Use `agents_get` with an `entry-name` to retrieve a specific agent definition.

## Available Agents

| Entry Name | Description |
|---|---|
| campaign-architect | Design and manage campaign arcs and world structure |
| character-builder | Guide character creation and optimization |
| encounter-designer | Design balanced tactical encounters |
| heroic-gm | Run table-ready GM workflows for sessions |
| rules-librarian | Look up and explain rules with precision |
